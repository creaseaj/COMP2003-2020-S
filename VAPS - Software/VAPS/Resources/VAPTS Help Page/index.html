Vulnerability Analysis and Pen-Testing Suite

Introduction

The Vulnerability Analysis and Pen-Testing Suite is a system testing kit designed for Small-to-Medium enterprises whos systems may be vulnerable. It searches for a range of vulnerabilities using traditional tools wrapped into one, user-friendly environment that outputs any results clearly and effectively.

VAPTS Functionality

ARP Scan

Our ARP (Address Resolution Protocol) viewer is a low-level network discovery tool, that gives users the ability to see the IP addresses and associated physical (MAC) addresses of devices on a network. In the screenshot below you can see VAPTS on the left giving the users a readout of a recent ARP scan, this can be compared to the default ARP command that ran through the command prompt on a windows machine on the right. Due to our built-in web scraping techniques, you will notice that VAPTS can identify the devices manufacturer. We felt it was important for the users to be able to quickly identify the devices and having more detailed information about said devices allows for that to happen.

Port Scan

Our port scanner allows the user to probe their network for open ports and gives a detailed readout of the port number, state, and description of what the port is used for, along with our recommendation on if the port should be open or closed. This can be seen at the bottom in the screenshot below, with easy to see, colour-coded indicators on security risk and the number of ports within each of the three categories shown clearly.

Password Strength Tester

VAPTS also features an integrated password strength tester to ensure that your passwords are up to professional security standards. The password strength tester uses specific criteria that, when met, will dramatically increase the security of a password.Once you enter a password, our tool will let you know which security criteria have been met and which still need to be added. It will also display the amount of time it would take for an attacker to crack your password. This means that the longer the time it would take to crack, the stronger the password is.
